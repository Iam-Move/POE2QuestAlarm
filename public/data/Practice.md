# POE2 퀘스트 네비게이션

Path of Exile 2 플레이어를 위한 **퀘스트 체크리스트 & 게임 오버레이** 도구입니다.

---

## 🛡️ 안전성

✅ **설치 불필요** - 웹 브라우저에서 실행
✅ **계정 안전** - 게임 파일을 건드리지 않음
✅ **데이터 보호** - 모든 정보는 브라우저에만 저장 (외부 전송 없음)

---

## ⚡ 빠른 시작

1. **필터 선택**: Regular / Semi-Strict / Uber / Custom
2. **퀘스트 체크**: 완료한 퀘스트를 클릭하여 체크
3. **오버레이 (선택)**: 게임 위에 띄워서 플레이

---

## 📋 필터 모드

| 필터 | 설명 |
|------|------|
| **Regular** | 모든 중요 퀘스트 (첫 플레이 추천) |
| **Semi-Strict** | 핵심 보상 퀘스트만 (효율 레벨링) |
| **Uber** | 필수 퀘스트만 (최속 클리어) |
| **Custom** | 직접 선택한 퀘스트만 (나만의 루트) |

---

## 🎮 오버레이 사용법

1. POE2를 **창 모드** 또는 **전체 창 모드**로 설정
2. 웹페이지에서 **'오버레이'** 버튼 클릭
3. 분리된 창을 게임 화면 위에 배치
4. 게임하면서 퀘스트 확인 & 체크

💡 오버레이는 항상 최상단에 표시됩니다 (Chrome/Edge 권장)

---

## ✏️ 편집 모드

**토글 스위치 ON** 시 사용 가능:

- **📝 내용 수정**: 퀘스트 이름, 보상, 메모 편집
- **↕️ 순서 변경**: 햄버거 아이콘(三) 드래그
- **➕ 퀘스트 추가**: Act 하단 "+ 새 퀘스트 추가" 버튼
- **🗑️ 퀘스트 삭제**: 각 퀘스트의 삭제 버튼
- **✓ Custom 반영**: Custom 필터에서 표시 여부 설정

모든 변경사항은 **자동 저장**됩니다.

---

## 🔗 공유 기능

**'설정 공유하기'** 버튼으로 진행 상황 공유:

1. **공유하기** 버튼 클릭
2. 제목과 설명 입력 (선택)
3. **링크 복사** 클릭
4. 친구에게 URL 전송

**받은 링크 열면**:
- 공유자의 필터 설정 & 완료 체크 상태 확인
- "적용하기"로 내 진행 상황에 반영 가능
- "취소"하면 내 설정 유지

💡 URL에 모든 정보가 압축되어 있어 서버 불필요!
